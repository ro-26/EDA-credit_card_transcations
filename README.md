### Featuresüí°:
- Utilizes a time-series data of credit card transactions along with information on Department, Division and Merchant and Description.
- Used python visualization libraries such as pandas plotly to visualize and gather insights.
- Time-series Anomaly Detection is performed using two techniques namely Isolation Forest and KMeans Clustering Algorithms.
- Further trained a KMeans model find similarities among different Departments in their temporal spending behavior and group them.
- And verified the same with the plot of dimensionally reduced data using IsoMap Dimensionality Reduction.
- Finally, a dashboard to showcase few of the insights gathered is made using Microsoft Power BI
